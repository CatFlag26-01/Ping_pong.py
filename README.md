Este programa fue hecho con Python y distintos módulos:

Pygame - Pygame es una biblioteca (o conjunto de módulos) de código abierto para el lenguaje de programación Python, diseñada específicamente para la creación y desarrollo de videojuegos 2D y aplicaciones multimedia
Ciclos - Los ciclos (o bucles) en Python son estructuras de control que permiten repetir un bloque de código varias veces mientras se cumple una condición o al iterar sobre una secuencia. Evitan la repetición manual de instrucciones, optimizando el flujo del programa, y se dividen principalmente en for (finito) y while (indefinido)
Clases - Las clases en Python son moldes o plantillas utilizadas para crear objetos, agrupando datos (atributos) y comportamientos (métodos) en una sola estructura. Permiten definir un nuevo tipo de objeto, facilitando la organización, reutilización y encapsulamiento del código en la programación orientada a objetos
PONG - Pong es un videojuego pionero de deportes, lanzado por Atari en 1972, basado en el tenis de mesa. Es considerado el primer videojuego comercialmente exitoso de la historia. Consiste en controlar una paleta (barra vertical) para golpear una pelota virtual de un lado a otro de la pantalla, intentando que el oponente no la devuelva

Que nos ayudaran a la funcionalidad común de dichas líneas de códigos del bloc de notas dando su aspecto determinado para funcionar correctamente
